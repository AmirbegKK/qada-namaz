# qada-namaz

Приложение для расчёта, планирования и ежедневного учёта къада-намазов.

## Что умеет

- рассчитывать ориентировочный долг по выбранному периоду;
- учитывать хайд как среднее число дней в месяц;
- показывать, сколько осталось и какая норма нужна на сегодня;
- быстро отмечать выполненные намазы;
- вести журнал прогресса;
- строить план по сроку или по дневной нагрузке;
- сохранять резервную копию данных и восстанавливать её при необходимости.

## Как пользоваться

1. Перейдите в `Расчёт и план`.
2. Укажите даты и, при необходимости, включите учёт хайда.
3. Нажмите `Пересчитать`.
4. Настройте план закрытия долга.
5. Вернитесь на `Главную` и отмечайте выполненные намазы каждый день.

## Удобство использования

- есть `Минимальный режим` для быстрого ежедневного отмечания;
- есть `Ночная тема`;
- одинаковые записи в журнале объединяются, чтобы список не разрастался;
- данные можно экспортировать в JSON как резервную копию.

## Важно

Расчёт в приложении ориентировочный.  
Если вы используете учёт хайда, итог зависит от указанного среднего числа дней в месяц.

Для надёжности полезно время от времени делать экспорт резервной копии.
